# react-boilerplate

boilerplate for Web Site using React


## Features

* Weboack 4
* Babel
* Sass(SCSS)
* Hot reload
* Webpack dev server
* ESLint
* Prettier


## Quick start

1. Clone this repo

```
git clone https://github.com/chocolat5/react-boilerplate.git <PROJECT_NAME>
```

2. Move to the appropriate directory

```
cd <PROJECT_NAME>
```

3. Install npm package

```
npm install --save-dev
```

4. Run the project

```
npm run start
```
